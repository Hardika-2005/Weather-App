# 🌦 Weather App

A simple and beautiful Weather Application that shows real-time weather details of any city using OpenWeather API.

---

## 🚀 Features

- 🌡 Shows temperature in Celsius
- 🌤 Displays weather condition (Clouds, Rain, Clear, etc.)
- 💨 Shows wind speed
- 🎨 Dynamic background based on weather
- 📱 Responsive design (works on mobile & desktop)

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API

---

## 📂 Project Structure

weather-app/
│── index.html
│── style.css
│── script.js
│── README.md

---

## ▶️ How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Enter a city name
4. Click **Get Weather**

---
